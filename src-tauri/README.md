# src-tauri — Tauri v2 Desktop Shell

Rust-based desktop shell wrapping the React frontend. Targets ~50MB RAM usage with zero FPS impact during gameplay.

## Commands

```bash
task dev:tauri     # Dev mode (frontend + backend + shell)
task build:tauri   # Production .exe for Windows distribution
```

See issue #4 (frontend setup) and #18 (build pipeline).
